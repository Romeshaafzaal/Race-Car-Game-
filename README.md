# Race-Car-Game-
A retro DOS-based racing game built in x86 Assembly (NASM), featuring enemy cars, collision detection, a bonus system, scrolling road, and a loading screen — developed as a COAL project at FAST NUCES.

## 🎮 Gameplay
- Navigate your car across **3 lanes**
- Avoid **randomly spawned enemy cars**
- Collect **bonus items** for extra points
- Score increases as enemy cars pass you

## ✨ Features
- Interrupt-driven keyboard & timer handling
- VGA text-mode graphics (80x25)
- Scrolling road with lane markers
- Random enemy spawning system
- Bonus item collection (+50 points)
- Pause menu (ESC) with exit confirmation
- Animated loading/cover screen with progress bar
- Game Over screen with final score

## 🕹️ Controls
| Key | Action |
|-----|--------|
| ← Left Arrow | Move left |
| → Right Arrow | Move right |
| ESC | Pause / Exit |
